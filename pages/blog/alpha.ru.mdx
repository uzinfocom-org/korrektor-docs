## Что нового

### Канализационная система

Система каналов была создана для того, чтобы быстрее создавать операционные системы без потери их эффективности и производительности. Мы считаем, что у каждой задачи есть свой собственный эффективный способ выполнения.
Чтобы достичь такой эффективности, мы должны создать специализированную операционную систему для каждого вида задач, которая будет сосредоточена только на ит. Тем не менее, вы, возможно, задавали себе вопрос, как мы могли бы создавать различные
изображения и не оказывать на них плохого влияния? Простое удаление ненужных инструментов и драйверов, оставив только важные вещи, повысит производительность системы. Кроме того, путем сборки готовых к производству конфигураций
мы можем сэкономить некоторое время пользователям на его настройке и повысить производительность. В настоящее время у нас есть только настольная версия, которая предназначалась для домашнего использования. В ближайшее время мы добавим еще 5 каналов.

### Зеркало

Мы создали наши собственные пакеты и сохранили их на нашем собственном зеркальном сервере. Чтобы использовать наше зеркало и установить наши пакеты, вам необходимо загрузить и установить наш открытый ключ gpg.

```shell
curl -Lo - "https://raw.githubusercontent.com/uzinfocom-org/keyring/main/xinux.gpg" \
    | pacman-key -a -

pacman-key --lsign-key info@uzinfocom.uz

pacman-key --init
pacman-key --populate xinux
```

затем вы можете добавить наш список зеркал в свой файл `/etc/pacman.conf`:

```yaml
[xinux]
SigLevel = Required DatabaseOptional
Server = https://mirror.xinux.uz/$repo/$arch/
Server = http://mirror.dc.uz/$repo/$repo/$arch/
Server = https://mirror.dc.uz/$repo/$repo/$arch/
```
